- `Views/RestWindow.xaml.cs` - 实现错误计数器和重置逻辑

**实现要点**：
```csharp

// 错误3次后显示重置按钮和错误提示
```

### 2.3 窗口位置记忆 ⭐

**需求描述**：
- 程序启动：始终居中，不记忆位置
- 托盘恢复：记忆隐藏前位置，恢复时显示在原位置
- 休息结束后：恢复到原来位置和大小

**涉及文件**：
- `ViewModels/MainViewModel.cs` - 添加位置记忆字段
- `Views/MainWindow.xaml.cs` - 在隐藏/显示时保存/恢复位置

**实现要点**：
```csharp
// 隐藏时保存位置
private void OnWindowClosing(object sender, CancelEventArgs e)
{
    _lastPosition = new Point(Left, Top);
    e.Cancel = true; // 取消关闭，改为隐藏
    Hide();
}

// 显示时恢复位置
public void ShowWindow()
{
    if (_lastPosition.HasValue)
    {
        Left = _lastPosition.Value.X;
        Top = _lastPosition.Value.Y;
    }
    Show();
}
```

**预计工时**：2小时

- 如果被覆盖，强制恢复置顶

**涉及文件**：
- `Views/RestWindow.xaml.cs` - 添加定时器定期检测并恢复Topmost

**实现要点**：
```csharp
// 定期检测置顶状态
_topmostTimer = new Timer(state =>
{
    Dispatcher.Invoke(() =>
    {
        if (!Topmost)
        {
            Topmost = true;
            Activate();
        }
    });
}, null, 1000, 1000); // 每秒检测一次
```
