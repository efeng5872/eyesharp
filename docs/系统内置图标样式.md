# Windows 系统内置图标样式参考

## SystemIcons 类提供的图标

| 图标名称 | 样式描述 | 适用场景建议 |
|---------|---------|-------------|
| **Application** | 默认应用程序图标（蓝色/灰色窗口） | 正常运行状态 |
| **Information** | 蓝色圆圈 + 白色字母 i | 正常运行状态 |
| **Shield** | 蓝色盾牌 | 正常运行状态（安全主题） |
| **WinLogo** | Windows 旗帜标志 | 正常运行状态 |

### 警告/暂停类图标
| 图标名称 | 样式描述 | 适用场景建议 |
|---------|---------|-------------|
| **Warning** | 黄色三角形 + 黑色感叹号 | 暂停状态（较醒目） |
| **Exclamation** | 黄色三角形 + 黑色感叹号（与Warning相同） | 暂停状态 |
| **Error** | 红色圆圈 + 白色 X | 暂停状态（较醒目） |
| **Question** | 蓝色圆圈 + 白色问号 | 暂停状态（询问用户） |

### 其他图标
| 图标名称 | 样式描述 | 适用场景建议 |
|---------|---------|-------------|
| **Hand** | 红色停止手势 | 暂停状态（停止主题） |
| **Asterisk** | 蓝色信息图标 | 正常运行状态 |

---

## 推荐组合方案

### 方案1：蓝绿正常 + 黄橙暂停（推荐）
- **正常运行**：`Application` 或 `Information`（蓝色系）
- **暂停状态**：`Warning` 或 `Exclamation`（黄色三角形）

**特点**：对比明显，黄色警告图标能清晰表达"暂停/注意"状态

### 方案2：统一蓝色系
- **正常运行**：`Information`（蓝色i）
- **暂停状态**：`Question`（蓝色?）

**特点**：风格统一，变化较 subtle

### 方案3：安全主题
- **正常运行**：`Shield`（蓝色盾牌）
- **暂停状态**：`Warning`（黄色警告）

**特点**："护眼助手"强调保护，盾牌形象贴切

## 图标预览方式

如果您想亲自查看这些图标样式，可以：

### 方法1：使用 C# 代码预览
```csharp
using System.Drawing;
using System.Windows.Forms;

// 显示图标
Icon icon = SystemIcons.Warning;
// 保存为图片查看
icon.ToBitmap().Save("warning.png");
```

### 方法2：Windows 系统自带
这些图标就是 Windows 系统消息框中使用的标准图标：
- 信息图标 → 蓝色i → 信息提示框
- 警告图标 → 黄色! → 警告提示框
- 错误图标 → 红色X → 错误提示框

## 建议

针对 EyeSharp 护眼助手，我推荐：

| 状态 | 图标 | 理由 |
|------|------|------|
| **正常运行** | `Information` (蓝色i) | 清新、专业、护眼主题蓝色 |
| **暂停提醒** | `Warning` (黄色!) | 醒目提醒用户注意（倒计时已暂停） |

这样可以形成明显的视觉区分，用户一眼就能看出程序当前状态。
