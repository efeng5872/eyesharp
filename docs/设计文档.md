## 3. 核心模块设计

### 3.1 配置管理模块 (ConfigService)

#### 3.1.1 配置文件结构

**文件路径**：
- 优先：`ApplicationDirectory/config.json`
- 备用：`%APPDATA%/eyesharp/config.json`

**JSON格式**：
```json
{
  "restIntervalMinutes": 30,
  "restDurationSeconds": 20,
  "isForcedMode": true,
  "restWindowMode": "black",
  "customImagePath": "",
  "passwordHash": "10000:a1b2c3d4:5e6f7g8h...",
  "autoStart": false,
  "logLevel": "INFO",
  "isPreReminderEnabled": true,
  "preReminderIntervals": [30, 10],
  "theme": "light"
}
```

#### 3.1.2 配置模型类

```csharp
public class AppConfig
{
    public int RestIntervalMinutes { get; set; } = 30;        // 1-120
    public int RestDurationSeconds { get; set; } = 20;        // 5-300
    public bool IsForcedMode { get; set; } = true;
    public string RestWindowMode { get; set; } = "black";     // "image" or "black"
    public string CustomImagePath { get; set; } = "";
    public string PasswordHash { get; set; } = "";            // format: "iterations:salt:hash"
    public bool AutoStart { get; set; } = false;
    public string LogLevel { get; set; } = "INFO";
    public bool IsPreReminderEnabled { get; set; } = true;
    public int[] PreReminderIntervals { get; set; } = { 30, 10 };
    public string Theme { get; set; } = "light";              // "light" or "dark"
}
```

#### 3.1.3 ConfigService设计

```csharp
public class ConfigService
{
    private readonly object _lock = new object();
    private readonly string _configPath;
    private readonly string _backupPath;

    public ConfigService(string exePath);

    // 加载配置
    public AppConfig LoadConfig();

    // 保存配置
    public bool SaveConfig(AppConfig config);

    // 恢复默认配置
    public AppConfig ResetToDefault();

    // 备份损坏文件
    private void BackupCorruptedFile(string originalPath);

    // 验证配置有效性
    private bool ValidateConfig(AppConfig config);

```

**线程安全**：使用`lock`确保配置读写操作的原子性。

**异常处理**：
- JSON反序列化异常 → 备份原文件 → 恢复默认配置 → 记录ERROR日志
- 文件IO异常 → 尝试备用路径 → 记录ERROR日志
- 配置验证失败 → 使用默认值替代 → 记录WARN日志

### 3.2 密码安全模块 (PasswordService)

#### 3.2.1 密码哈希算法

使用**PBKDF2-HMAC-SHA256**算法：

- **迭代次数**：10,000次
- **盐值长度**：16字节（随机生成）
- **哈希长度**：32字节（SHA-256输出）
- **存储格式**：`iterations:salt:hash`（Base64编码）

**示例**：
```
10000:YWJjZDEyMzQ1Njc4OWFiY2Q=:YTFlZjJnM2g0aTVqNmU3Zzg5aTBramwxbTJuM280cDVR==
```

#### 3.2.2 PasswordService设计

```csharp
public class PasswordService
{
    private const int Iterations = 10000;
    private const int SaltSize = 16;
    private const int HashSize = 32;

    // 生成密码哈希
    public string HashPassword(string password);

    // 验证密码
    public bool VerifyPassword(string password, string storedHash);

    // 生成随机盐值
    private byte[] GenerateSalt();

    // 使用PBKDF2计算哈希
    private byte[] ComputeHash(string password, byte[] salt, int iterations, int hashSize);
}
```

**安全性考虑**：
- 每个密码使用独立的随机盐值
- 盐值与哈希结果一起存储，支持验证
- 迭代10,000次增加暴力破解难度
- 使用HMAC-SHA256避免长度扩展攻击

##### ViewModel实现

```csharp
/// <summary>
/// 修改密码对话框ViewModel
/// </summary>
public partial class ChangePasswordDialogViewModel : ObservableObject
{
    private readonly PasswordService _passwordService;
    private readonly ConfigService _configService;

    [ObservableProperty]
    private string _oldPassword = "";

    [ObservableProperty]
    private string _newPassword = "";

    [ObservableProperty]
    private string _confirmPassword = "";

    [ObservableProperty]
    private string _errorMessage = "";

    [ObservableProperty]
    private string _passwordLengthHint = "密码长度：4-20个字符";

    [ObservableProperty]
    private bool _isOkButtonEnabled = false;

    public ChangePasswordDialogViewModel(PasswordService passwordService,
                                        ConfigService configService)
    {
        _passwordService = passwordService;
        _configService = configService;
    }

    // 密码输入变化时验证
    partial void OnOldPasswordChanged(string value)
    {
        ValidatePasswords();
    }

    partial void OnNewPasswordChanged(string value)
    {
        ValidatePasswords();
    }

    partial void OnConfirmPasswordChanged(string value)
    {
        ValidatePasswords();
    }

    // 验证密码
    private void ValidatePasswords()
    {
        // 检查旧密码是否已输入
        if (string.IsNullOrWhiteSpace(OldPassword))
        {
            ErrorMessage = "";
            IsOkButtonEnabled = false;
            return;
        }

        // 检查新密码长度
        if (NewPassword.Length < 4 || NewPassword.Length > 20)
        {
            ErrorMessage = "密码长度必须在4-20字符之间";
            IsOkButtonEnabled = false;
            return;
        }

        // 检查新密码和确认密码是否一致
        if (NewPassword != ConfirmPassword)
        {
            ErrorMessage = "两次输入的密码不一致";
            IsOkButtonEnabled = false;
            return;
        }

        // 所有验证通过
        ErrorMessage = "";
        IsOkButtonEnabled = true;
    }

    // 修改密码
    [RelayCommand]
    private async Task ChangePasswordAsync()
    {
        try
        {
            // 验证旧密码
            var config = _configService.LoadConfig();
            bool isOldPasswordValid = _passwordService.VerifyPassword(OldPassword,
                                                                               config.PasswordHash);

            if (!isOldPasswordValid)
            {
                ErrorMessage = "旧密码不正确";
                return;
            }

            // 生成新密码哈希
            string newPasswordHash = _passwordService.HashPassword(NewPassword);

            // 保存到配置文件
            config.PasswordHash = newPasswordHash;
            _configService.SaveConfig(config);

            LogService.LogInfo("密码修改成功");

            // 关闭对话框
            // 在Code-behind中调用 DialogResult = true
        }
        catch (Exception ex)
        {
            LogService.LogError($"修改密码失败: {ex.Message}", ex);
            ErrorMessage = "密码修改失败，请重试";
        }
    }
}
```

##### Code-Behind实现

```csharp
/// <summary>
/// 修改密码对话框 Code-Behind
/// </summary>
public partial class ChangePasswordDialog : Window
{
    private readonly ChangePasswordDialogViewModel _viewModel;

    public ChangePasswordDialog(PasswordService passwordService,
                                ConfigService configService)
    {
        InitializeComponent();

        _viewModel = new ChangePasswordDialogViewModel(passwordService, configService);
        DataContext = _viewModel;

        // 订阅ViewModel属性变化
        _viewModel.PropertyChanged += (s, e) =>
        {
            if (e.PropertyName == nameof(ChangePasswordDialogViewModel.ErrorMessage))
            {
                ErrorMessage.Text = _viewModel.ErrorMessage;
            }
            else if (e.PropertyName == nameof(ChangePasswordDialogViewModel.IsOkButtonEnabled))
            {
                OkButton.IsEnabled = _viewModel.IsOkButtonEnabled;
            }
        };
    }

    private void OnPasswordChanged(object sender, RoutedEventArgs e)
    {
        // 触发ViewModel验证
        // ViewModel的属性变更会自动更新UI
    }

    private void OnCancelClick(object sender, RoutedEventArgs e)
    {
        // 关闭对话框，不保存
        DialogResult = false;
        Close();
    }

    private async void OnOkClick(object sender, RoutedEventArgs e)
    {
        // 调用ViewModel的修改密码方法
        await _viewModel.ChangePasswordAsync();

        // 如果没有错误，关闭对话框
        if (string.IsNullOrEmpty(_viewModel.ErrorMessage))
        {
            // 显示成功提示（可选）
            MessageBox.Show("密码已成功修改", "成功",
                          MessageBoxButton.OK,
                          MessageBoxImage.Information);

            DialogResult = true;
            Close();
        }
    }
}
```

### 3.3 定时器模块 (TimerService)

#### 3.3.1 设计方案：单TimerService + 状态机

**核心思想**：整个应用程序只使用一个TimerService实例，通过状态机管理定时器在不同场景下的行为。

- ✅ 更容易维护和调试
- ✅ 减少资源占用（仅一个DispatcherTimer）

#### 3.3.4 TimerService完整设计

```csharp
/// <summary>
/// 定时器服务（单例，全局唯一）
/// </summary>
public class TimerService : ObservableObject
{
    private readonly DispatcherTimer _timer;
    private DateTime _startTime;
    private TimeSpan _targetInterval;
    private TimerState _state = TimerState.Running;

    public TimerService()
    {
        _timer = new DispatcherTimer
        {
            Interval = TimeSpan.FromMilliseconds(100) // 100ms刷新一次UI
        };
        _timer.Tick += OnTimerTick;
    }

    // ========== 属性 ==========

    /// <summary>
    /// 当前状态
    /// </summary>
    public TimerState State
    {
        get => _state;
        private set
        {
            if (_state != value)
            {
                var oldState = _state;
                _state = value;
                OnPropertyChanged();
                StateChanged?.Invoke(this, new TimerStateChangedEventArgs(oldState, value));
            }
        }
    }

    /// <summary>
    /// 当前剩余时间
    /// </summary>
    public TimeSpan RemainingTime { get; private set; }

    /// <summary>
    /// 是否运行中（Running 或 Resting）
    /// </summary>
    public bool IsRunning => State == TimerState.Running || State == TimerState.Resting;

    /// <summary>
    /// 是否暂停
    /// </summary>
    public bool IsPaused => State == TimerState.Paused;

    // ========== 方法 ==========

    /// <summary>
    /// 开始主窗口倒计时
    /// </summary>
    public void StartMainCountdown(TimeSpan interval)
    {
        _targetInterval = interval;
        _startTime = DateTime.Now;
        State = TimerState.Running;
        _timer.Start();
        LogService.LogInfo($"主窗口倒计时开始: {interval.TotalMinutes} 分钟");
    }

    /// <summary>
    /// 开始休息窗口倒计时
    /// </summary>
    public void StartRestCountdown(TimeSpan interval)
    {
        _targetInterval = interval;
        _startTime = DateTime.Now;
        State = TimerState.Resting;
        // _timer已经在运行，不需要重新Start
        LogService.LogInfo($"休息窗口倒计时开始: {interval.TotalSeconds} 秒");
    }

    /// <summary>
    /// 结束休息，返回主窗口倒计时
    /// </summary>
    public void EndRest()
    {
        State = TimerState.Running;
        LogService.LogInfo("休息结束，返回主窗口倒计时");
    }

    /// <summary>
    /// 暂停倒计时（仅在Running状态有效）
    /// </summary>
    public void Pause()
    {
        if (State == TimerState.Running)
        {
            _timer.Stop();
            State = TimerState.Paused;
            LogService.LogInfo("主窗口倒计时已暂停");
        }
    }

    /// <summary>
    /// 恢复倒计时（仅在Paused状态有效）
    /// </summary>
    public void Resume()
    {
        if (State == TimerState.Paused)
        {
            _startTime = DateTime.Now - (_targetInterval - RemainingTime); // 修正启动时间
            _timer.Start();
            State = TimerState.Running;
            LogService.LogInfo("主窗口倒计时已恢复");
        }
    }

    /// <summary>
    /// 停止定时器
    /// </summary>
    public void Stop()
    {
        _timer.Stop();
        State = TimerState.Running; // 重置为初始状态
    }

    /// <summary>
    /// 重置主窗口倒计时间隔（仅在Running或Paused状态下调用）
    /// </summary>
    /// <param name="newInterval">新的间隔时间</param>
    public void ResetMainInterval(TimeSpan newInterval)
    {
        if (State == TimerState.Running)
        {
            // 运行状态下立即重置
            _targetInterval = newInterval;
            _startTime = DateTime.Now;
            RemainingTime = newInterval;
            OnPropertyChanged(nameof(RemainingTime));
            LogService.LogInfo($"主窗口倒计时间隔重置: {newInterval.TotalMinutes} 分钟");
        }
        else if (State == TimerState.Paused)
        {
            // 暂停状态下更新间隔，恢复时使用新值
            _targetInterval = newInterval;
            RemainingTime = newInterval;
            OnPropertyChanged(nameof(RemainingTime));
            LogService.LogInfo($"暂停状态下修改间隔为: {newInterval.TotalMinutes} 分钟（恢复后生效）");
        }
        else if (State == TimerState.Resting)
        {
            // 休息状态下不应该调用此方法（UI已禁用输入框）
            LogService.LogWarn($"休息状态下尝试重置间隔，操作已忽略");
            throw new InvalidOperationException("休息状态下不能重置主窗口间隔");
        }
    }

    /// <summary>
    /// 当前配置的休息时长（从配置文件读取）
    /// </summary>
    public TimeSpan ConfiguredRestDuration { get; set; } = TimeSpan.FromSeconds(20);

    /// <summary>
    /// 修改休息时长配置（下次休息生效）
    /// </summary>
    /// <param name="newDuration">新的休息时长</param>
    public void UpdateRestDuration(TimeSpan newDuration)
    {
        ConfiguredRestDuration = newDuration;
        LogService.LogInfo($"休息时长已更新: {newDuration.TotalSeconds} 秒（下次休息生效）");
    }

    // ========== 事件 ==========

    /// <summary>
    /// 倒计时结束事件
    /// </summary>
    public event EventHandler<CountdownElapsedEventArgs> MainCountdownElapsed;

    /// <summary>
    /// 休息倒计时结束事件
    /// </summary>
    public event EventHandler<CountdownElapsedEventArgs> RestCountdownElapsed;

    /// <summary>
    /// 状态改变事件
    /// </summary>
    public event EventHandler<TimerStateChangedEventArgs> StateChanged;

    // ========== 私有方法 ==========

    private void OnTimerTick(object sender, EventArgs e)
    {
        // 计算真实剩余时间（避免DispatcherTimer累积误差）
        var elapsed = DateTime.Now - _startTime;
        RemainingTime = _targetInterval - elapsed;

        if (RemainingTime <= TimeSpan.Zero)
        {
            RemainingTime = TimeSpan.Zero;

            if (State == TimerState.Running)
            {
                // 主窗口倒计时结束
                MainCountdownElapsed?.Invoke(this, new CountdownElapsedEventArgs());
            }
            else if (State == TimerState.Resting)
            {
                // 休息倒计时结束
                RestCountdownElapsed?.Invoke(this, new CountdownElapsedEventArgs());
            }
        }

        // 通知UI更新
        OnPropertyChanged(nameof(RemainingTime));
    }

    // ========== 清理资源 ==========

    public void Cleanup()
    {
        _timer?.Stop();
        _timer.Tick -= OnTimerTick;
    }
}

/// <summary>
/// 定时器状态改变事件参数
/// </summary>
public class TimerStateChangedEventArgs : EventArgs
{
    public TimerState OldState { get; }
    public TimerState NewState { get; }

    public TimerStateChangedEventArgs(TimerState oldState, TimerState newState)
    {
        OldState = oldState;
        NewState = newState;
    }
}

/// <summary>
/// 倒计时结束事件参数
/// </summary>
public class CountdownElapsedEventArgs : EventArgs
{
}
```

```csharp
public class MainViewModel : ObservableObject
{

    private readonly Window _mainWindow;
    private RestWindow _restWindow;

    private TimeSpan _remainingTime;

    public MainViewModel(Window mainWindow, TimerService timerService)
    {

        _timerService = timerService;

        // 订阅定时器事件
        _timerService.MainCountdownElapsed += OnMainCountdownElapsed;
        _timerService.StateChanged += OnTimerStateChanged;

        // 绑定剩余时间
        _timerService.PropertyChanged += (s, e) =>
        {
            if (e.PropertyName == nameof(TimerService.RemainingTime))
            {
                RemainingTime = _timerService.RemainingTime;
            }
        };
    }

    private void OnMainCountdownElapsed(object sender, CountdownElapsedEventArgs e)
    {
        LogService.LogInfo("主窗口倒计时结束，触发休息提醒");

        // 隐藏主窗口
        _mainWindow.Hide();

        // 切换定时器到休息状态
        var config = _configService.LoadConfig();
        _timerService.StartRestCountdown(TimeSpan.FromSeconds(config.RestDurationSeconds));

        // 显示休息窗口
        _restWindow = new RestWindow(_timerService);
        _restWindow.ShowDialog();
    }

    private void OnTimerStateChanged(object sender, TimerStateChangedEventArgs e)
    {
        LogService.LogInfo($"定时器状态: {e.OldState} → {e.NewState}");
    }

    // 暂停按钮命令
    [RelayCommand]
    private void PauseResume()
    {
        if (_timerService.State == TimerState.Running)
        {
            _timerService.Pause();
        }
        else if (_timerService.State == TimerState.Paused)
        {
            _timerService.Resume();
        }
    }
}
```

#### 3.3.6 时间修正机制

为避免DispatcherTimer的累积误差，使用**真实时间计算**：

```csharp
// 记录启动时间
private DateTime _startTime;

// Tick事件中计算真实剩余时间
private void OnTimerTick(object sender, EventArgs e)
{
    var elapsed = DateTime.Now - _startTime;
    RemainingTime = _targetInterval - elapsed;

    if (RemainingTime <= TimeSpan.Zero)
    {
        RemainingTime = TimeSpan.Zero;
        // 触发倒计时结束事件
    }
}

// 暂停后恢复时修正启动时间
public void Resume()
{
    // 重新计算启动时间，确保剩余时间不变
    _startTime = DateTime.Now - (_targetInterval - RemainingTime);
    _timer.Start();
    State = TimerState.Running;
}
```

**优势**：
- 每100ms刷新一次UI，保证倒计时显示流畅
- 基于`DateTime.Now`计算剩余时间，避免累积误差
- 实际精度±15ms，远超需求（±100ms）

- ✅ State属性变更自动触发PropertyChanged事件

```csharp
    /// 系统托盘工具提示文字
    /// </summary>
    [ObservableProperty]
    private string _trayToolTipText = "护眼助手 - 休息倒计时中";

    // ========== 构造函数 ==========

    {
        _timerService = timerService;
        _trayIconService = trayIconService;

        // 订阅定时器状态改变事件
        _timerService.StateChanged += OnTimerStateChanged;

        // 初始化UI状态
        UpdateUIState(_timerService.State);

    /// <summary>
    /// 定时器状态改变时同步所有UI
    /// </summary>
    private void OnTimerStateChanged(object sender, TimerStateChangedEventArgs e)
    {
        UpdateUIState(e.NewState);
        LogService.LogInfo($"UI状态同步: {e.OldState} → {e.NewState}");
    }

    /// <summary>
    /// 根据定时器状态更新所有UI组件
    /// </summary>
    private void UpdateUIState(TimerState state)
    {
        switch (state)
        {
            case TimerState.Running:
                // 主窗口UI
                PauseResumeButtonText = "暂停提醒";
                StatusText = "运行中";

                // 系统托盘UI
                TrayPauseResumeMenuText = "暂停提醒";
                TrayToolTipText = "护眼助手 - 休息倒计时中";
                _trayIconService.UpdateStatus(isPaused: false);
                break;

            case TimerState.Paused:
                // 主窗口UI
                PauseResumeButtonText = "恢复提醒";
                StatusText = "已暂停";

                // 系统托盘UI
                TrayPauseResumeMenuText = "恢复提醒";
                TrayToolTipText = "护眼助手 - 休息倒计时已暂停";
                _trayIconService.UpdateStatus(isPaused: true);
                break;

            case TimerState.Resting:
                // 休息状态下，主窗口隐藏，托盘保持运行状态
                PauseResumeButtonText = "暂停提醒";
                StatusText = "休息中";

                // 系统托盘UI（休息期间托盘图标保持正常状态）
                TrayPauseResumeMenuText = "暂停提醒";
                TrayToolTipText = "护眼助手 - 休息时间中";
                _trayIconService.UpdateStatus(isPaused: false);
                break;
        }

        // 通知所有属性更新（CommunityToolkit.Mvvm自动处理）
        LogService.LogDebug($"UI状态已更新: {state}");
    }

    // ========== 命令：主窗口暂停/恢复 ==========

    /// <summary>
    /// 主窗口暂停/恢复按钮命令
    /// </summary>
    [RelayCommand]
    private void PauseResumeFromMainWindow()
    {
        if (_timerService.State == TimerState.Running)
        {
            _timerService.Pause();
            LogService.LogInfo("用户从主窗口暂停倒计时");
        }
        else if (_timerService.State == TimerState.Paused)
        {
            _timerService.Resume();
            LogService.LogInfo("用户从主窗口恢复倒计时");
        }
    }

    // ========== 命令：托盘暂停/恢复 ==========

    /// <summary>
    /// 系统托盘菜单暂停/恢复命令

    {

        {
            _timerService.Pause();

        {
            _timerService.Resume();
            LogService.LogInfo("用户从系统托盘恢复倒计时");
        }
    }

    // ========== 清理资源 ==========

    public void Cleanup()

    }
```

**场景3：两个入口同时操作（冲突处理）**

```

2. 用户立即点击托盘"暂停" → State: Paused → Paused（无变化）

- TimerService.Pause()检查：if (State == Running) { State = Paused; }
- 多次暂停操作：第一次生效，后续无操作（幂等性）
- TimerService.Resume()检查：if (State == Paused) { State = Running; }
- 多次恢复操作：第一次生效，后续无操作（幂等性）

结论：通过State检查保证操作的幂等性，无需额外处理
```

### 3.3.9 锁屏处理策略设计

#### 3.3.9.1 策略概述

系统支持三种锁屏处理策略，通过 `LockScreenBehavior` 配置控制：

| 策略 | 配置值 | 主倒计时期间锁屏行为 | 休息期间锁屏行为 |
|------|--------|---------------------|-----------------|
| **正常显示** | normal | 与锁屏无关，主倒计时到点即进入休息流程 | 休息倒计时继续，正常完成休息 |
| **暂停倒计时** | pause | 主倒计时期间锁屏立即暂停，解锁后自动恢复 | 休息倒计时继续，正常完成休息 |
| **跳过休息** | skip | 仅当主倒计时到点时处于锁屏：不弹休息窗口，进入等待解锁 | 休息倒计时继续，正常完成休息 |

#### 3.3.9.2 统一锁屏架构设计（方案B）

**核心思想**：
- 锁屏状态由 `ILockStateService` 统一提供（单一事实来源）
- 锁屏策略由 `TimerService` 统一决策（单一状态机）
- `MainViewModel` 仅负责 UI 显示，不再直接处理锁屏策略分支

**关键结构**：

```csharp
public interface ILockStateService
{
    bool IsLocked { get; }
    event EventHandler<LockStateChangedEventArgs> LockStateChanged;
}

public class TimerService : ITimerService
{
    private readonly ILockStateService _lockStateService;
    private bool _isWaitingForUnlock;
    private bool _pausedByLockStrategy;

    // 锁屏状态变化统一入口
    private void OnLockStateChanged(object? sender, LockStateChangedEventArgs e)
    {
        HandleLockStateChanged(e.IsLocked);
    }
}
```

**策略行为（按最终需求）**：

1. **normal**
   - 与锁屏无关，主倒计时到点即触发休息窗口；
   - 休息期间锁屏不影响休息倒计时。

2. **pause**
   - 主倒计时期间锁屏：立即暂停；
   - 解锁：自动恢复；
   - 休息期间锁屏不影响休息倒计时。

3. **skip**
   - 仅当主倒计时到点时 `IsLocked=true`：跳过本次休息，进入“等待解锁”；
   - 解锁后触发 `SkipRest`，直接启动新主倒计时；
   - 到点时未锁屏则正常显示休息窗口。

#### 3.3.9.3 UI状态联动

`TimerService` 提供 `WaitForUnlockChanged` 事件：
- 进入等待解锁：主界面状态文案显示“等待解锁”；
- 退出等待解锁并恢复运行：状态文案恢复“运行中”。

```csharp
private List<string> ShuffleImages(List<string> images)
{
    var random = new Random();
    var result = images.ToList();

    for (int i = result.Count - 1; i > 0; i--)
    {
        int j = random.Next(i + 1);
        (result[i], result[j]) = (result[j], result[i]);
    }

    return result;
}
```

##### 自动关闭机制设计

**设计决策**：✅ **实现自动关闭功能**

**理由**：
- 提升用户体验，无需用户点击"确定"
- 避免对话框干扰用户操作
- 符合现代UI/UX标准

**实现方式**：
```csharp
public partial class SingleInstanceDialog : Window
{
    private const int AutoCloseDelaySeconds = 5;
    private DispatcherTimer _autoCloseTimer;

    public SingleInstanceDialog()
    {
        InitializeComponent();
        StartAutoCloseTimer();
    }

    private void StartAutoCloseTimer()
    {
        _autoCloseTimer = new DispatcherTimer
        {
            Interval = TimeSpan.FromSeconds(1) // 每秒更新一次倒计时
        };
        _autoCloseTimer.Tick += OnTimerTick;
        _autoCloseTimer.Start();
    }

    private int _countdown = AutoCloseDelaySeconds;

    private void OnTimerTick(object sender, EventArgs e)
    {
        _countdown--;

        if (_countdown <= 0)
        {
            // 倒计时结束，自动关闭对话框
            _autoCloseTimer.Stop();
            DialogResult = true;
            Close();
        }
        else
        {
            // 更新倒计时显示
            CountdownText.Text = $"{_countdown}秒后自动激活已运行窗口...";
        }
    }

    private void OnOkButtonClick(object sender, RoutedEventArgs e)
    {
        // 用户点击"确定"按钮，立即关闭对话框
        _autoCloseTimer?.Stop();
        DialogResult = true;
        Close();
    }

    protected override void OnClosed(EventArgs e)
    {
        _autoCloseTimer?.Stop();
        _autoCloseTimer.Tick -= OnTimerTick;
        base.OnClosed(e);
    }
}
```

```csharp
{
    // 验证范围
    if (_restIntervalMinutes < 1 || _restIntervalMinutes > 120)
    {
        IntervalApplyStatusText = "超出范围（1-120）";
        return;
    }

    // 应用到定时器
    _timerService.ResetMainInterval(TimeSpan.FromMinutes(_restIntervalMinutes));

    // 更新已应用的值
    _appliedRestIntervalMinutes = _restIntervalMinutes;

    // 清除待应用状态
    HasPendingChanges = false;
    IntervalApplyStatusText = $"已应用: {_restIntervalMinutes} 分钟";

    // 保存配置
    _configService.SaveConfig(CreateConfig());

    LogService.LogInfo($"用户应用新的休息间隔: {_restIntervalMinutes} 分钟");
}

// 修改检测
partial void OnRestIntervalMinutesChanged(int value)
{
    // 检查是否有待应用的更改
    HasPendingChanges = (value != _appliedRestIntervalMinutes);

    if (HasPendingChanges)
    {
        IntervalApplyStatusText = "点击【应用】按钮生效";
    }
    else
    {
        IntervalApplyStatusText = "";
    }
}

// 定时器状态改变时更新控件可编辑性
private void OnTimerStateChanged(object sender, TimerStateChangedEventArgs e)
{
    IsTimeIntervalEditable = (e.NewState != TimerState.Resting);
```

## 5. 异常处理设计

### 5.1 异常处理策略

| 异常类型              | 处理方式                          | 用户反馈               |
| ----------------- | ----------------------------- | ------------------ |
| **配置文件损坏**        | 备份原文件 → 恢复默认配置                | 对话框提示 + WARN日志    |
| **配置文件读写失败**      | 尝试备用路径 → 使用默认配置              | ERROR日志           |
| **图片加载失败**        | 回退到内置图片                       | 无（静默处理）+ WARN日志 |
| **密码验证失败**        | 显示错误提示                         | UI错误提示 + INFO日志  |
| **定时器异常**         | 重启定时器 → 记录日志                   | ERROR日志           |
| **窗口显示异常**        | 强制激活主窗口 → 记录日志                | ERROR日志           |
| **未处理异常**         | 全局异常捕获 → 记录日志 → 显示友好错误提示 → 优雅退出 | ERROR日志 + 对话框     |

### 5.2 全局异常捕获

```csharp
// App.xaml.cs
public partial class App : Application
{
    protected override void OnStartup(StartupEventArgs e)
    {
        // UI线程异常
        DispatcherUnhandledException += OnDispatcherUnhandledException;

        // 非UI线程异常
        AppDomain.CurrentDomain.UnhandledException += OnUnhandledException;

        base.OnStartup(e);
    }

    private void OnDispatcherUnhandledException(object sender, DispatcherUnhandledExceptionEventArgs e)
    {
        LogService.LogError($"UI线程异常: {e.Exception.Message}", e.Exception);

        // 显示友好错误对话框
        MessageBox.Show($"程序发生错误：{e.Exception.Message}\n\n详细信息已记录到日志文件。",
                       "错误",
                       MessageBoxButton.OK,
                       MessageBoxImage.Error);

        e.Handled = true; // 防止程序崩溃
    }

    private void OnUnhandledException(object sender, UnhandledExceptionEventArgs e)
    {
        if (e.ExceptionObject is Exception ex)
        {
            LogService.LogError($"非UI线程异常: {ex.Message}", ex);
        }
    }
}
```

**TimerService实现**：
```csharp
public bool IsPreReminderEnabled { get; set; } = true;
public int[] PreReminderIntervals { get; set; } = { 30, 10 };
private HashSet<int> _triggeredReminders = new HashSet<int>();

private void CheckPreReminders(TimeSpan remaining)
{
    var remainingSeconds = (int)remaining.TotalSeconds;

    foreach (var interval in PreReminderIntervals)
    {
        if (remainingSeconds <= interval && !_triggeredReminders.Contains(interval))
        {
            _triggeredReminders.Add(interval);
            OnPreReminder(new PreReminderEventArgs
            {
                SecondsUntilRest = remainingSeconds,
                Message = $"将在{interval}秒后开始休息"
            });
        }
    }
}

public event EventHandler<PreReminderEventArgs>? PreReminder;
```

**MainViewModel订阅**：

| **多显示器环境**        | 低    | 使用虚拟屏幕范围（SystemParameters.VirtualScreenLeft/Top/Width/Height） |
| **配置文件并发冲突**      | 低    | 使用lock保护，UI线程操作为主         |
| **定时器精度不足**       | 低    | 记录启动时间，计算真实剩余时间，避免累积误差  |
| **密码破解风险**        | 低    | PBKDF2+随机盐值+10,000次迭代      |
| **日志文件过大**        | 低    | NLog自动归档，限制归档数量           |
